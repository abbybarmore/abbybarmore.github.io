Refer to the previous code description. Name this code block ‘clean’. In this block, separate the numbers in the score column to W and L and do this by using the dash in between them. What is before the dash will be in the ‘W’ column and the numbers after the dash will be in the ‘L’ column. And then rename the W column ‘TeamScore’ and the L column ‘OpponentScore’.

This separates puts the Team Score and Opponent Score in different columns and makes it easy to use when coding.

clean <- logs %>% separate(Result, into=c("W", "L"), sep="\\(") %>% 
rename('Outcome' = 'W', 'Score' = 'L') %>%
mutate(Score=gsub(")", "", Score)) %>%
separate(Score, into=c("W", "L"), sep="\\-") %>%
rename('TeamScore' = 'W', 'OpponentScore' = 'L') %>% mutate(TeamScore = as.numeric(TeamScore), OpponentScore = as.numeric(OpponentScore))

Create a new data frame called worsegames using the information from the clean data frame that was just created. And then create or mutate a new column titled ‘Differential’ which is TeamScore minus the OpponentScore. And then filter this by teams with differentials that are greater than 65 points.

worsegames <- clean %>% mutate(
  Differential = TeamScore - OpponentScore) %>%
  filter(Differential > 65)

With this information, let’s create a scatterplot to see what we’ve found. Create a chart with points from the clean data frame and plot the data of Team Score and Opponent Score on the x and y-axis, respectively. Make there points grey. Along with this, create a line with the same information. This line will illustrate the average of all the teams.

ggplot() + 
  geom_point(data=clean, aes(x=TeamScore, y=OpponentScore), color='grey') +
  geom_smooth(data=clean, aes(x=TeamScore, y=OpponentScore), method="lm")
